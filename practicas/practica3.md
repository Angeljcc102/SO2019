#Conceptos

1) Proceso

+ Instancia de un propgrama.

+ Tiene tres partes:

++ Stack: variables y las llamadas a funciones.

++ Heap: memoria dinámica (malloc o new).

++ Código


+ Se crea mediante dos llamadas a sistema:

++ fork clona

++ exec cambia el código


+ Tiene un estado:

++ Corriendo

++ Durmiendo

++ Espera

++ Completado

++ Zombie
